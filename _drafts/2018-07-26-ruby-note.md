### Instant Arrays and Hashes from method call

```Ruby
class Document
  def add_authors(names)
    @authors += "#{names.join(' ')}"
  end
end
```

```Ruby
Document.new.add_authors( [ 'Strunk', 'White' ] )
```

```Ruby
class Document
  def add_authors(*names)
    @authors += "#{names.join(' ')}"
  end
end
```

```Ruby
Document.new.add_authors('Strunk', 'White')
```
